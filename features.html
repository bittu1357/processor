In this section you will find how to perform preprocessing on this application.

This application is for data Preprocessing. Here user upload your data set wich you want to perform preprocessing on it.
      In this application you will find features like,
      1.  Data Description
      2.  Handling Null Values
      3.  Encoding Categorical Columns
      4.  Feature Scaling
      5.  Download the pre-processed data set

1. Data Description

       In Data Description user will find three options

         1. Show the properties of each column:
                  In this option user can see the properties of each column like, count, unique values, mean value, most freq value, etc.

         2. Show the properties of specific column
                  In this option user can see the properties for perticular column.

         3. Show the data set
                  In this option user can see the whole data set.

2. Handling Null Values

       In Handling Null Values user will find three options

         1. Remove/Delete column:
                  In this option user can delete or remove the perticular column by selecting column name from drop down menu.

         2. Show the numbers of null values
                  In this option user can see total number of null values all columns.

         3. Fill null Values
                  In this option user can fill the null values with mean or median or mode by selecting column from the drop down menu.

3. Encoding Categorical Columns

       In Encoding Categorical Columns user will find two options

         1. Show the Categorical Columns:
                  In this option user can see the how many numbers of caltegorical columns and its unique values.

         2. Select Column To Perform OneHotEncoding
                   In this option user selects the Categorical column name to performe the One-Hot-Encoading operation .

4. Feature Scaling

       In Feature Scaling user will find four options

         1. Perform Nornalization (MinMax Scaler) For A Whole Datase:
                  In this option user can apply Normalization using MinMax-Scaler for whole dataset.

         2. Perform Nornalization (MinMax Scaler) For A particular Column
                  In this option user can apply Normalization using MinMax-Scaler for perticular column of dataset.

         3. Perform Standardization(Standard Scaler) For Whole Data Set
                  In this option user can apply Standardization using Standerd-Scaler for whole dataset.

         4. Perform Standardization(Standard Scaler) For A particular Column
                  In this option user can apply Standardization using Standerd-Scaler for perticular column of dataset.

5. Download the pre-processed data set

       By selecting this option user can download pre-processed dataset.
